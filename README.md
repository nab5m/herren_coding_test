# 헤렌 코딩 테스트

작성자: 김준영

## 브랜치 관리
- master(=main) <br>
: 프로덕션 서버, 실제 배포에 사용할 브랜치 <br>

- develop <br>
: 개발 서버 <br>

## 목표
1. black이라는 파이썬 코드 포매터 사용해보기
2. swagger로 문서화하기
3. 내 aws 계정에 배포하기
4. 코테 합격하기

## 실행방법
```shell script
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```
